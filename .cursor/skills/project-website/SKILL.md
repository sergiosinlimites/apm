# Skill: Sitio Web del Proyecto

## Cuándo Usarla

Al crear o modificar el sitio web del proyecto (entregable 11.5 del EDT).

## Ubicación

Todo el sitio web vive en `website/`. El sitio se despliega en GitHub Pages o similar.

## Secciones Mínimas

1. **Inicio**: Nombre del proyecto, descripción breve, equipo
2. **Proceso**: Descripción del proceso de manufactura, layout de planta
3. **Productos**: Los 3 productos con sus características
4. **Simulación**: Resultados AS-IS vs TO-BE
5. **Control**: Sistema de control, Grafcet, Ladder
6. **SCADA**: Pantallas HMI, arquitectura
7. **Robótica**: Celda robotizada, RobotStudio
8. **Resultados**: Indicadores, OEE, comparativos
9. **Equipo**: Integrantes con foto/nombre/rol

## Checklist

- [ ] Responsive (mobile + desktop)
- [ ] Paleta de colores del proyecto aplicada
- [ ] Contenido en español
- [ ] Imágenes optimizadas con alt text
- [ ] Navegación clara entre secciones
- [ ] Desplegable en GitHub Pages (solo archivos estáticos)
